ISO 26262 ASIL Levels

ISO 26262 defines ASIL QM, A, B, C, D. There is no ASIL-E in the standard. (If a stakeholder asks for “ASIL A–E”, treat E as out-of-scope / map to ASIL-D + additional OEM criteria.)
