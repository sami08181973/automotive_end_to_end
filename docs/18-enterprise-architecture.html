EA Layer Stack

L5 – Business / Capability

Safe mobility, comfort UX, continuous feature delivery, fleet learning, regulatory compliance (ISO 26262, ISO/SAE 21434, ASPICE).

L4 – Application / Function Domains

Infotainment, Connectivity, ADAS, Body, Powertrain, Chassis, Functional features (ACC/AEB/OTA…), Digital Twin apps.

L3 – Data Architecture

Signal dictionary, edge buffers, encrypted uplink, analytics marts, twin fidelity metrics, GDPR anonymization policies.

L2 – Software & Integration

AUTOSAR Classic/Adaptive, SOME/IP, DDS, MQTT, ARA::COM, CI/CD release trains, OTA lifecycle.

L1 – Technology / E/E

Zonal controllers, Vehicle HPC, Automotive Ethernet, CAN/CAN-FD/LIN, hypervisor, QNX safety OS, HSM.

L0 – Security & Safety Cross-Cut

Secure boot, gateway firewall, ASIL allocation, health monitoring, CRC/E2E protection, ISO 21434 controls.
